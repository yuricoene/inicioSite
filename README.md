Interface Rede de Jogos

Projeto de extensão desenvolvido para praticar conceitos de Front-End. O objetivo foi criar uma landing page moderna, imersiva e totalmente responsiva para o público gamer.

Funcionalidades
- Interface em modo escuro (Dark Theme).
- Design responsivo para dispositivos móveis.
- Menu de navegação fixo com efeitos de hover.
- Ícones estilizados com filtros CSS.

Tecnologias
- HTML5
- CSS3 (Flexbox, Variáveis, Filtros)
- Google Fonts (Roboto)
